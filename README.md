# Snowflake Performance

## Running the app locally

Set the following fields using environment variable or in `snowflake_util.py`

    SNOWFLAKE_ACCOUNT
    SNOWFLAKE_DB
    SNOWFLAKE_SCHEMA
    SNOWFLAKE_WAREHOUSE
    SNOWFLAKE_ROLE
    SNOWFLAKE_USERNAME
    SNOWFLAKE_PASSWORD

To run the app: 
    
    streamlit run App.py
